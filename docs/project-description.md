#### Flow:

1. Set a Parameter of what we consider to be correct (Research paper's…)
2. Data will be searched over the Parameter and result is out

#### 1→ How to select a Parameter?

thing or webspace which is globally believed in, or in the particular field

(considering the parameter to be research Papers for understanding)

#### 2→ Options:

use Perplexity API for finding such research Papers — scrape them — give any LLM to parse and provide result

**or**

(Do search ourself at place of perplexity)
